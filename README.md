Color Meaning Finder (Tkinter App)

A simple Python GUI application built using Tkinter.
It displays the psychological meaning of colors and changes the window background accordingly.

Features

Dropdown selection using Combobox

Background color updates dynamically

Meaning of each color displayed clearly

Minimal and easy-to-use UI

How to Run
python colour_changer.py

Requirements

Python 3.x

Tkinter (preinstalled with most Python versions)
